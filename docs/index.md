# Adani Data Platform

Welcome to the Adani Group Data Platform documentation portal.

## Overview

The Adani Data Platform spans **8 business units**, each with dedicated **Dev** and **Prod** Databricks workspaces. Every workspace hosts:

- **3 Genie Agents** - Natural language AI interfaces to query business data
- **5 Data Products** - Curated, governed datasets for consumption
- **2 Dashboards** - Visual analytics for business stakeholders
